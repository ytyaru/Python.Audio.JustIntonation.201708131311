# このソフトウェアについて

純正律の7音を算出した。

# 対象ファイル名

ファイル名|説明
----------|----
playJustIntonation.py|純正律の音を再生してファイル出力する
MusicTheory/temperament/JustIntonation.py|純正律により基音から7音を算出する

# 実行

```sh
$ python playJustIntonation.py 
```

# 課題

中間の5音も算出したい。計算方法がよくわからない。
